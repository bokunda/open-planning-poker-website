Open Planning Poker

Welcome to Open Planning Poker!

Open Planning Poker is a collaborative agile estimation tool designed to streamline your sprint planning sessions. It's an engaging and effective way for development teams to estimate the effort required for each user story or task in a project.

GitHub page

How it works:

The team gathers to discuss and estimate the complexity of user stories.
Each team member privately selects an estimation value using a deck of cards.
The selected values are revealed simultaneously, fostering discussion and convergence.
Repeat until a consensus is reached for each user story.

Make your planning sessions more collaborative, efficient, and fun with our Open Planning Poker app.
